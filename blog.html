Important FAQ

What is the Document Object Model(DOM)?

Document Object Model(DOM) is a programming interface for web pages. It give access to all the HTML elements and developer can change the elements, add new elements and remove elements. It makes HTML elements as a tree of objects. So the DOM is very essential for web development it allows dynamic interaction of web pages in real-time.

How do you select an element from the DOM?

There are many methods in javascript that helps to select elements from the DOM. The most common methods are getElementById(), getElementsByTagName(), getElementsByClassName(), querySelector(), querySelectorAll().

What is event delegation in the context of the DOM, and Why is it useful?

Event delegation is basically handle events in the Dom. Events can be added to a parent element instead of adding to every single element.
It reduces the number of event listeners in the code, improves performance, simplifies code management.

How do you manipulate an element's attributes and styles using the DOM?

To manipulate an element's attributes, we can use getAttribute(), setAttribute(), removeAttribute() or directly modify properties like id, class etc.
we can modify inline styles by using "style" property or use classList to mange css classes.
